SoftOnt

een zelf gemaakte bedrijf oprichten en in dat bedrijf kunnen werken met de 3 geleerde lagen:
  • Presentatie Laag
    de code die de gebruiker gaat zien. bv.Facebooks homepage
  • Buisness Laag
    de code dat regels bevat die specifiek zijn voor het probleem waarvoor de applicatie is gebouwd bv. analyses, berekeningen, ect...
  • DataAcces Laag
    is een laag die vereenvoudigde toegang verschaft tot gegevens die zijn opgeslagen in jow database
